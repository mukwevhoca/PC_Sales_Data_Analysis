# PC_Sales_Data_Analysis

#Project Overview:
This project analyses PC sales data to uncover business insights such as trends, top-performing products, and sales performance by region. 

_ _ _

#Objectives:
-Analyse total revenue
-Identify top-selling PC models
-Evaluate sales performance by region and salesperson
-Calculate profit and profit margins

_ _ _

#Tools Used:
-SQL Server (SSMS)

_ _ _

#What I learned:
-Writing efficient SQL queries for data analysis
-Using aggregation and grouping functions
-Translating business questions into SQL logic
